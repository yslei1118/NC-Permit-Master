# 北卡Permit考试刷题App

这是一个基于R Shiny开发的北卡驾照考试刷题应用，帮助你高效备考！

## ✨ 功能特性

### 📚 题目分类
- **常规选择题** (140题): 驾照规定、交通规则、安全驾驶等
- **管制标志** (27题): STOP、YIELD、限速、禁止标志等
- **警告标志** (27题): 黄色菱形警告标志识别
- **高速公路标志** (9题): 州际公路、路线标记、服务设施等

### 🎯 练习模式
- **顺序练习**: 按题号依次练习
- **随机练习**: 随机打乱题目顺序
- **错题重刷**: 专门练习答错的题目
- **题目列表**: 查看所有题目状态，从任意题开始

### 💡 智能功能
- ✅ **实时反馈**: 答题后立即显示正确答案和详细解析
- ✅ **错题本**: 自动记录错题，答对后自动移除
- ✅ **收藏功能**: 收藏重点题目，随时复习
- ✅ **进度追踪**: 实时显示完成进度（已做/总数）
- ✅ **统计分析**: 多维度统计学习情况和正确率
- ✅ **图片支持**: 73道题含标志图片

## 🚀 安装依赖

首先确保已安装R (>= 4.0)，然后安装依赖包：

```r
install.packages(c("shiny", "shinydashboard", "dplyr"))
```

## 💻 启动应用

### 方法1：使用RStudio（推荐）
1. 在RStudio中打开 `app.R`
2. 点击右上角的 "Run App" 按钮

### 方法2：使用命令行
```bash
cd permit_app
Rscript run_app.R
```

或者：
```bash
Rscript -e "shiny::runApp('.', launch.browser = TRUE)"
```

## 📖 使用说明

### 开始练习
1. 在主页选择题目类型（常规选择题/管制标志/警告标志/高速公路标志）
2. 选择练习模式（顺序/随机/错题重刷/题目列表）
3. 点击选项A/B/C/D提交答案
4. 查看答案解析，点击"下一题"继续

### 功能说明
- **收藏题目**: 点击⭐按钮收藏重点题目
- **查看错题**: 点击"错题本"查看所有答错的题目
- **统计分析**: 查看各分类的完成进度和正确率
- **重置进度**: 点击左侧边栏的"重置进度"清空所有记录

### 退出练习
- 点击"退出"按钮会自动返回到对应的题目分类页面
- 点击"完成练习"会显示完成提示并返回

## 📊 题库说明

当前题库包含**203道题目**：

- 管制标志：27题
- 警告标志：27题

### 图片说明
- 73道题目包含标志图片
- 所有图片已整理到 `images/` 文件夹的对应子目录
- 图片格式：PNG/JPEG

### 进度计算
- 进度 = 做过的题目数 / 该部分题目总数
- 例如：`进度：25/140 (17.9%)` 表示已完成25题，总共140题

## 📝 数据文件

### questions.csv
题库数据文件，包含所有题目。CSV格式：

```csv
id,type,sign_name,question,option_a,option_b,option_c,option_d,answer,explanation,handbook_ref,image_ref
1,general,NA,题目内容,选项A,选项B,选项C,选项D,B,解析内容,chapter_1,NA
```

字段说明：
- `id`: 题目编号（唯一）
- `type`: 题目类型（general/regulatory/warning/highway）
- `sign_name`: 标志名称（标志类题目）
- `question`: 题目内容
- `option_a/b/c/d`: 四个选项
- `answer`: 正确答案（A/B/C/D）
- `explanation`: 答案解析
- `handbook_ref`: 参考章节
- `image_ref`: 图片路径（相对于images/）

### user_progress.rds
用户答题记录文件，包括：
- 答题历史
- 错题记录
- 收藏题目
- 练习进度

删除此文件可以重置所有进度。
